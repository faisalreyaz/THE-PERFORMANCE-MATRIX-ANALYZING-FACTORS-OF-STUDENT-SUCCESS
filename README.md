# 📊 Student Performance Analysis Dashboard

## 📌 Overview

This project explores the key factors that drive student success using a comprehensive dataset of **10,000 students**. The interactive dashboard highlights relationships between study habits, attendance, stress levels, learning styles, and academic performance.

The goal of this analysis is to uncover actionable insights that educators, institutions, and students can use to improve outcomes.

---

## 🚀 Features

* 📈 **Key Metrics Overview**

  * Total Students
  * Average Weekly Study Hours
  * Attendance Rate
  * Assignment Completion Rate
  * Exam Scores

* 🎯 **Performance Breakdown**

  * Exam scores distribution by grade (A–D)
  * Student performance categories:

    * High Performers
    * Average Performers
    * At-Risk Students

* 🧠 **Behavioral Insights**

  * Stress level vs. performance trends
  * Learning styles vs. exam scores

* 🔍 **Interactive Filters**

  * Gender-based filtering
  * Performance tier selection

## 📂 Dataset

The dataset includes the following key fields:

* `Student_ID`
* `Gender`
* `Study_Hours_Per_Week`
* `Attendance_Rate (%)`
* `Assignment_Completion_Rate (%)`
* `Exam_Score (%)`
* `Stress_Level` (Low, Medium, High)
* `Preferred_Learning_Style`
* `Final_Grade`

---

## 📊 Key Insights

* 📚 **Study Hours Matter**
  Students study an average of ~27 hours/week, strongly correlating with better scores.

* 🏫 **Attendance & Assignments Drive Success**
  Attendance (~75%) and assignment completion (~75%) show a clear impact on exam performance.

* ⚠️ **Stress Negatively Impacts Performance**
  Higher stress levels are associated with lower performance outcomes.

* 🎧 **Learning Style Has Minor Impact**
  Auditory and kinesthetic learners slightly outperform others, but differences are marginal.

* 🥇 **Majority Are High Performers**
  Over half of the students fall into the high-performing category.
